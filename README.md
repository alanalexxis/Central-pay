## Run Locally

### Clone the project

```sh
git clone https://github.com/alanalexxis/Central-pay.git
```

### Go to the project directory

```sh
cd central-pay
```

### Install dependencies

```sh
npm install
```

### Start the server

```sh
npm run dev
```

### Open the web app in your browser

[http://localhost:3000](http://localhost:3000)
